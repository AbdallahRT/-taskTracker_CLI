# Task Tracker CLI App

Task Tracker is a CLI application used to track and manage your tasks.  
More information can be found here: [Task Tracker Roadmap](https://roadmap.sh/projects/task-tracker)

## Installation Instructions

1. Clone the GitHub repository to your machine.
2. Make sure you have **npm** installed on your machine.
3. Run the following command from the repository to install the package globally:

```bash
npm install -g .
